# Factory Pro

A modern, responsive industrial and manufacturing company website built with HTML, CSS, and JavaScript.

## 📋 Overview

Factory Pro is a professional business website template designed for manufacturing companies, industrial firms, and construction businesses. It features a modern design with smooth animations and responsive layouts.

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom styles with Bootstrap 5.3
- **JavaScript** - Vanilla JS with jQuery
- **Bootstrap 5** - CSS framework
- **Owl Carousel** - Carousel/slider component
- **Font Awesome 7** - Icons
- **AOS** - Animate On Scroll library

## 📁 Project Structure


## ✨ Features

- Responsive design (mobile-friendly)
- Smooth scroll animations
- Interactive navigation with mobile menu
- Image sliders/carousels
- Contact form
- Modern UI with hover effects
- Cross-browser compatible

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rmsuvajit18/Factory-Pro.git
   ```

2. **Open in browser:**
   - Simply open `index.html` in any web browser
   - Or use a local server (e.g., Live Server in VS Code)

## 📄 Pages

| Page | Description |

## 🔧 Customization

- Edit `css/style.css` for custom styles
- Modify `js/script.js` for custom JavaScript
- Replace images in the `images/` folder
- Update favicon in `favicon/` folder
