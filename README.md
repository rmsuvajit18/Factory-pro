# Factory Pro

A modern industrial and manufacturing company website built with HTML, CSS, and JavaScript.

## Overview

Factory Pro is a professional business website designed for industrial and manufacturing companies. It features a modern design with responsive layouts, smooth animations, and interactive elements.

## Features

- **Responsive Design** - Fully responsive across all devices
- **Modern UI** - Clean and professional interface
- **Interactive Elements** - Smooth animations and transitions
- **Multiple Pages** - Complete website with multiple sections
- **Carousel Sliders** - Image and content sliders using Owl Carousel
- **Contact Form** - Functional contact form with validation
- **Font Awesome Icons** - Rich icon set for UI elements
- **AOS Animations** - Scroll-triggered animations

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Home page with hero section and services |
| `about.html` | Company information and team |
| `project.html` | Portfolio of completed projects |
| `pricingplan.html` | Service pricing plans |
| `cms.html` | Content management page |
| `contact.html` | Contact information and form |
| `latestpost.html` | Blog/News section |

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Styling with Bootstrap 5.3
- **JavaScript** - Interactive functionality
- **Bootstrap** - CSS framework
- **Owl Carousel** - Carousel slider
- **AOS** - Animation library
- **Font Awesome** - Icon library

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/rmsuvajit18/Factory-pro.git
   ```

2. Open `factory-pro/index.html` in your browser
