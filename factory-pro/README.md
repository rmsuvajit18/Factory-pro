# Factory Pro

A modern, responsive business website for industrial and manufacturing services.

## Overview

Factory Pro is a professional business website template designed for manufacturing, industrial, and service-based companies. It features a modern design with responsive layouts, smooth animations, and interactive elements.

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom styling with Bootstrap 5
- **JavaScript** - Interactive functionality
- **Libraries**:
  - Bootstrap 5.3.8 (CSS framework)
  - Owl Carousel (carousel/slider)
  - Font Awesome 7.0.1 (icons)
  - AOS (Animate On Scroll)

## Features

- Fully responsive design
- Smooth scroll animations (AOS)
- Interactive navigation with dropdown menus
- Owl carousel sliders
- Contact form
- Font Awesome icons
- Modern, clean UI

## Getting Started

1. Clone or download the project
2. Open `index.html` in a web browser
3. Navigate using the menu links

No build process or server required - simply open the HTML files in a browser.
